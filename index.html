The Basics

Web developement is the process of building a website. To build a website you start by building code. The programming language we started with is HTML. HTML stands for "Hypertext Markup Language."

The basic structure looks like this, which includes a Doctype, html stags, head, and a body. See how they nest inside eachother?

<html>

<head>
</head>

<body>
</body>

</html>

Then we learned about heading tags, or <hn> tags, n being anywhere between 1-6. These are used for heading or titling a section.

Lets say you wanted to put something down as a note for yourself, that you don't need to render on your webpage? You can use a comment, which looks like this:

We can also group items together using <div> tags, for either layout or styling. Each of the examples on this page were grouped together with div tags.

We can also add specific styling to elements inline using a <span> tag.

<p>Only one word is <span>red</span>.</p>

Only one word is red.

These are good to style specific parts of an element and not split up an element.

CSS aka "Flavor"

Congrats! We've now learned enough material to make a page from 1996! But if you really want to give your friend's GeoCities' webpage a run for it's money, we should add some style to it. We can use CSS, or Cascasding Style Sheets, to achieve this. Styling could be anything from placement on a webpage, to changing the color on the font. Using CSS we can draw attention to specific parts of a page, to make it asthetically appealing and easy to navagate information. They can be applied to any element, class, or ID on a page. Here is an example:

p {
font-family: helvetica;
}

This code is put on a separate page, and it can be applied to multiple elements, or just one specific element. In the example above all the paragraph tags will now be in the Helvetica font.

Each element can be thought of as a box. Each box has four sections. The content, the padding between the content and the border, the border, and the margin around the border. See the example below (note that the dashed line represents the end of the margin space).

The box model is an effective tool when it comes to grouping elements together, and spacing out other elements accordingly.

HTML5 & CSS3

There are several versions of HTML and CSS, and the current versions nearing completion are HTML5 and CSS3. With these versions we are able to make code easier to understand and have some creative new tools in order to animate code, without the aid of javascript.

Some of the new tags created for HTML5 are <header>, <nav>, <section>, <aside>, and <footer>. They are just specified div names, but they work just the same as any normal <div>. The only difference is that they are are semantic, meaning they relate better with tradition printing and layouts. The semantic tags were used in generating this page.

With CSS3 we can use new properties to add unique effects to text, as well as elements. A couple of them are called transforms and animations.
